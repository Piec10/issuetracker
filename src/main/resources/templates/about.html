Issue Tracker

Web application for managing projects created by many users

Issue Tracker is an application created for educational purposes, modeled on such software as Jira or Trello. It allows to create projects, add issues to them and manage them: different priority levels, editing, closing a completed issue. It allows to add multiple users with different access levels to the project. The application uses a MySQL database. Responsive frontend ensures proper display also on mobile devices.
